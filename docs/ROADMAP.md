# ShipTrust Roadmap

## Product direction

ShipTrust is a guided trust-readiness scanner for indie SaaS builders, AI tool creators and small teams.

The product helps builders understand how trustworthy their product looks before users decide to sign up, pay, or leave.

## Current milestone

### v0.2 - ShipTrust guided trust scanner

- clean dark/red guided scanner experience
- AI-style Home entry screen
- Light Check flow
- Deep Research guided assistant
- animated analysis screen
- results dashboard with score, trust gaps and developer tasks
- trust page draft direction
- markdown report export

## Next milestones

- better use of product descriptions

### v0.4 - Better generated trust page/report

- stronger generated trust page sections
