# Build in Public

ShipTrust is being built as a public product experiment for indie SaaS trust readiness.

## What is being built

ShipTrust helps app builders understand how trustworthy their product looks before users decide to sign up, pay, or leave.

The current MVP is a guided scanner that turns a product profile into trust gaps, developer tasks, a trust page draft and a markdown report.

## Why

Early-stage builders often ship features before they know whether the product feels credible enough for real users or first B2B customers.

ShipTrust focuses on visible customer-facing trust signals instead of heavy enterprise workflows.
