# Changelog

## v0.2 - ShipTrust guided trust scanner

### What changed

- Rebuilt the product from Compliance Guard/Regulato direction into ShipTrust.
- Added clean dark/red guided scanner experience.
- Added AI-style Home entry screen.
- Added guided trust check flow.
- Added animated analysis screen.
- Added results dashboard with trust score, trust gaps and developer tasks.
- Added trust page and markdown report direction.
- Cleaned UI copy and removed legacy positioning.

### Why it matters

Early-stage builders need a clear path from product description to actionable trust-readiness output.

### User problem solved

Builders can see what users may question before they sign up, pay, or start a first B2B conversation.

### What comes next

Stronger guided profile detection, better generated trust page sections and saved/shareable scans.
